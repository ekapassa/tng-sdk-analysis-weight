5GTANGO SDK's Analyze Weight Tool


tng-sdk-analyze-weight Home Page

- URI: tng-sdk-analyze-weight/api/weight/v1/home
   Method: GET
   Purpose: Visit the home page

Get weight for VNFs in a Network Service

- URI: tng-sdk-analyze-weight/api/weight/v1/"nsd_uuid"
   Method: GET
   Purpose: Get correlated parameters for the VNFs in a NS

Upload a data set

- URI: tng-sdk-analyze-weight/api/weight/v1/train/new/vnf/"vnf_type" -F 'file=@FileName.csv'
   Method: POST/GET
   Purpose: Upload a new dataset for a new VNF and store it locally

Get weight factors for a VNF list

- URI: tng-sdk-analyze-weight/api/weight/v1/vnftype?vnf_type=?,?,?,?,?
   Method: GET
   Purpose: Get Json result with the weight factors for all the provided VNF types

Get supported VNF types

- URI: tng-sdk-analyze-weight/api/weight/v1/mgmt/knownvnfs
   Method: GET
   Purpose: Get a list with the supported-known VNF types

Get unsupported-unknown VNF types

- URI: tng-sdk-analyze-weight/api/weight/v1/mgmt/unknownvnfs
   Method: GET
   Purpose: Get a list with the unsupported-unknown VNF types

Get correlation figure for a VNF

- URI: tng-sdk-analyze-weight/api/weight/v1/mgmt/"vnf_type"
   Method: GET
   Purpose: Get Correlation Figure for the corresponding VNF
