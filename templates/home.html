5GTANGO SDK's Analysis Tool

tng-sdk-analysis-weight Home Page

- URI: /api/weight/v1
  Method: GET
  Purpose: Visit the home page (API Specification)

Train

- URI: /api/weight/v1/train
  Method: GET
  Purpose: Start Training for the local data sets

Get weight for VNFs parameters

- URI: /api/weight/v1/
  Method: GET
  Purpose: Get correlated parameters for the VNFs in a NS

Update a data set

- URI: /api/weight/v1/train/new/vnf/ -F 'file=@FileName.csv'
  Method: POST/GET
  Purpose: Upload a new dataset for a new VNF and store it locally

Get weight factors for a VNF list

- URI: /api/weight/v1/vnftype?vnf_type=?,?,?,?,?
  Method: GET
  Purpose: Get Json result with the weight factors for all the provided VNF types

Get supported VNF types

- URI: /api/weight/v1/mgmt/knownvnfs
  Method: GET
  Purpose: Get a list with the supported-known VNF types
